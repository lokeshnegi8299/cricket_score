# Cricket Score (Android)

This project starts directly on a **Match Setup** screen (not the default "Hello Android" template).

## Open in Android Studio

1. Open Android Studio.
2. Click **Open** and select this folder: `cricket_score`.
3. Let Gradle sync complete.
4. Run on emulator/device.

## If you still see "Hello Android"

You are likely running a different project/module or an old app build.

- Confirm package is `com.example.cricketscore`.
- Confirm launcher activity is `com.example.cricketscore.MainActivity`.
- In Android Studio, use **Build > Clean Project** then **Build > Rebuild Project**.
- Uninstall the old app from the emulator/device and run again.

## What you should see first

- Team 1 name + Team 2 name
- Number of players per team (dynamic player name fields)
- Total overs
- Start Match button that navigates to scoring screen
